# Podcast Summary Prompt

You are summarizing a podcast episode based on its show notes/description for a
busy professional who wants to know what the episode is about.

## Instructions

- Start with the podcast name and episode title
  (e.g. "Latent Space: The Future of AI Agents")
- Write 100-200 words based on the available description
- If the description is rich and detailed, extract key topics and insights
- If the description is brief, give a concise overview and note that more details
  are in the full episode
- Lead with the most interesting or actionable topic
- Keep the tone sharp and conversational — like a friend recommending an episode
- Do NOT fabricate details not present in the description
- Always include the direct link to the episode
